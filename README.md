## 🚀 Projects

### 💊 [PRIS — Pharmaceutical Regulatory Intelligence System](https://github.com/yugalrade19/PRIS--)
> Predicting whether a drug is **Regulated** or **Non-Regulated** using real-world pharmaceutical data.

- 🧹 Handled class imbalance (74% vs 26%) using **SMOTE**
- 🤖 Compared **Logistic Regression, Random Forest, XGBoost** with hyperparameter tuning
- 📊 Used **SHAP** for model explainability — understanding *why* the model makes each prediction
- 🏆 Final model: Tuned **XGBoost** with GridSearchCV
- 🔧 Tools: `Python` `Pandas` `Scikit-learn` `XGBoost` `SHAP` `Seaborn`

---

### 🧠 [3D Organoid Topology Intelligence](https://github.com/yugalrade19/3D-Organoid-Topology-Intelligence)
> Analyzing the **3D spatial structure of cancer organoids** using topology features from microscopy data.

- 🔬 Worked with biomedical data — cell density, neighbor distances, ellipsoid axes, Ripley-corrected metrics
- 📐 Explored 3D anisotropy, local crowding, and spatial elongation of cells
- 🤖 Applied **Logistic Regression, Random Forest, XGBoost** with Stratified K-Fold cross-validation
- 📊 Statistical testing with **Mann-Whitney U** to compare experimental conditions
- 🔧 Tools: `Python` `Pandas` `NumPy` `Scikit-learn` `XGBoost` `SciPy` `Matplotlib`
